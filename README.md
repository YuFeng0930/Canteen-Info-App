# Real-time Canteen Information System

### Objectives:
1. Store and display stall information
1. Store and display stall menus 
1. Display stall information and menus based on current system date and time
1. Display stall information and menus based on user defined date and time
1. Calculate estimated waiting time for the stall by asking user to enter the number of people in the queue
1. Allow to check the operating hours for all stalls
